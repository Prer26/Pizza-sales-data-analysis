## 📌 Project Overview
This project analyzes pizza sales data to understand revenue trends, customer preferences, and category performance using Power BI.

## 🎯 Objectives
- Identify best selling pizza categories  
- Analyze revenue by size and type  
- Understand order patterns  
- Create interactive business dashboard

## 🛠 Tools Used
- Power BI  
- DAX  
- Power Query  
- Data Modeling

## 📊 Key Insights
- Classic category generated highest revenue  
- Large size pizzas contributed most sales  
- Peak orders observed on weekends

## 📁 Files
- Pizza_Sales.pbix – Power BI dashboard  
- pizza_sales.csv – dataset  
- dashboard_preview.png – screenshot

## 🚀 How to Use
1. Download the .pbix file  
2. Open in Power BI Desktop  
3. Refresh data source
